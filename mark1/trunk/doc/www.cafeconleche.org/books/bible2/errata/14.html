Corrections to Chapter 14 of The XML Bible, CSS Style Sheets

p. 364: In the second code fragment on the page monospaced should be monospace. That is, it should read:

POEM > REFRAIN > VERSE {font-family: Courier, monospace}

p. 369: In the last line of p. 369 font-style should be font-weight. That is, it should read:

STANZA:first-child { font-weight: bold }
