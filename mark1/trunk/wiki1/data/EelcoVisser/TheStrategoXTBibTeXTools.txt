------+++ Abstract

This paper provides documentation for the Stratego/XT <nop>BibTeX Tools,
a collection of tools for processing <nop>BibTeX bibliography files. The
main tools in the collection are <code>bib-to-html</code> for generating a
sectioned publication list from a <nop>BibTeX file according to
selection criteria and a layout template, and
<code>aux-to-bib</code> for extracting a <nop>BibTeX file based on the
citations in a <nop>LaTeX document.
