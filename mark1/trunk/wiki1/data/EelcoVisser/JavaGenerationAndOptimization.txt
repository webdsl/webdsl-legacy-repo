Implement a framework for the generation and transformation of Java programs.

This involves

	* Create a good syntax definition of Java in SDF
		* pretty-printer
		* concise abstract syntax
	* Develop utitlities 
		* free variable analysis
		* renaming of bound variables
		* etc.
	* Implement several transformations; some ideas
		* visualization of Java programs (e.g., transformation to UML)
		* generation 
		* aspect weaving
