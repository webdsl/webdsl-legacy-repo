-------++ Projects

These are software projects that I am involved in:

--------+++ Transformation

	* [[http://www.stratego-language.org][Stratego]] is a language for the specification of program transformation systems based on the paradigm of rewriting strategies.

	* [[http://www.syntax-definition.org][SDF2]] is a modular formalism for syntax definition that integrates lexical and context-free syntax.

	* [[http://www.stratego-language.org/Stratego/StrategoXT][Stratego/XT]] is a bundle of tools for program transformation built around Stratego, SDF2 and the ATerm library.

	* [[http://www.stratego-language.org/Stratego/JavaFront][Java-Front]] provides a syntax definition for Java and [[http://www.stratego-language.org/Stratego/JavaBorg][JavaBorg]] implements sample domain-specific language extensions for Java.

	* [[http://www.stratego-language.org/Stratego/JavaBorg][JavaBorg]]: example domain-specific language embeddings 

	* [[http://www.octave-compiler.org][Octave Compiler]]

	* [[http://www.stratego-language.org/Stratego/BibtexTools][BibTeX Tools]] is a set of tools for transforming <nop>BibTeX files, in particular production of html bibliographies and extraction of bib files based on a .aux file.

	* [[http://www.stratego-language.org/Tiger][Tiger in Stratego]] is a collection of components for the compilation and transformation of Tiger programs designed for the purpose of education and demonstration of program transformation techniques.

	* [[http://www.codeboost.org][CodeBoost]] is a transformation system for C++
