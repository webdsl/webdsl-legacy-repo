Dependencies

The following is a list of dependencies for this project. These dependencies are required to compile and run the application:

Artifact ID Type Version URL Comment
commons-io jar 1.1 http://jakarta.apache.org/commons/io/
portlet-api jar 1.0 http://portals.apache.org/pluto Required only when using FileUpload in a portlet environment.
servlet-api jar 2.3 http://java.sun.com/products/servlet/
junit jar 3.8.1 http://www.junit.org/ Test Only - required only for running the FileUpload unit tests.
maven-changelog-plugin plugin 1.9.1 http://maven.apache.org/reference/plugins/changelog/ Site Only - v1.9.1 (minimum) required for building the FileUpload Site documentation.
maven-xdoc-plugin plugin 1.9.2 http://maven.apache.org/reference/plugins/xdoc/ Site Only - v1.9.2 (minimum) required for building the FileUpload Site documentation.
